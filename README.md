# C++Bignum，一个高精度模板
###### 正在完善中。
## 1. 使用方法
### 1.1 变量定义
**类型名是``bignum``。**

你可以像下面那样定义它：
```
bignum n;
```
非常简单。
### 1.2 输入与输出
此模板重载了输入输出流的运算符，**直接使用``cin ``和``cout``输入输出即可。**

像这样
```
bignum n;
cin>>n;
cout<<n<<endl;
```
### 1.3 变量赋值
**直接使用赋值运算符``=``赋值即可。**
像这样：
```
bignum a,b,t;
cin>>a>>b;
t=a;
a=b;
b=t;
cout<<a<<b;
```
### 1.4 取正负值
直接像int一样使用``+````-``运算符即可。

像这样
```
bignum a;
cin>>a;
cout<<-a<<endl;
```
